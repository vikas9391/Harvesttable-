# HarvestTable — Moroccan Artisan Herbs & Teas

A full-stack e-commerce site built with **React + TypeScript + Vite + TailwindCSS**.

## Stack
- **Frontend**: React 18, TypeScript, Vite 5, TailwindCSS 3
- **Routing**: React Router v6
- **Backend**: Django 5 + Django REST Framework (in `/backend`)
- **Fonts**: Playfair Display (headings) + DM Sans (body)

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:3000)
npm run dev

# Build for production
npm run build
```

## Pages
| Route | Page |
|-------|------|
| `/` | Home |
| `/products` | Shop — filter by category/dietary |
| `/product/:slug` | Product detail |
| `/gift-builder` | 3-step gift box builder |
| `/about` | Our Story |
| `/checkout` | 3-step checkout |
| `/admin` | Admin dashboard (login: admin / admin123) |
| `/admin/products` | Product management |
| `/admin/orders` | Order management |
| `/admin/customers` | Customer list |
| `/admin/settings` | Store settings |

## Color Theme
Matches the live site: dark stone/brown (`stone-950`) background, amber/gold accents, terracotta CTAs.
